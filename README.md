# CopiCost — Copilot Request Cost Calculator

Track the real cost of every GitHub Copilot request, directly in VS Code.

CopiCost parses the **Agent Debug Logs** (JSONL) generated by Copilot and calculates the cost of each LLM call based on token usage and model pricing.

## Features

### Status Bar — Monthly Budget Tracker
A permanent indicator in the status bar shows your spending for the current month as a percentage of the $20/month Copilot budget.

```
$(pulse) $0.042 · 0.21% of $20/mo
```

Icons change as you approach the limit:
- `$(pulse)` — Normal (< 50%)
- `$(warning)` — Moderate (50–80%)
- `$(flame)` — High (> 80%)

### Sidebar — Session Explorer
The CopiCost panel in the Activity Bar lists all your sessions with:
- Color-coded cost indicators (green → yellow → orange → red)
- Relative timestamps ("5m ago", "2h ago")
- Per-request breakdown with model name and % of session cost
- Rich tooltips with full token/cost tables

### Click on a Request — Detail Panel
A dedicated webview showing:
- Large cost amount with color coding
- Token breakdown (fresh input, cached input, output)
- Visual cost distribution bar
- Cache impact analysis (how much caching saved you)
- Cost perspective (extrapolation to hourly/daily/monthly rates)

### Full Report
Command `CopiCost: Show Cost Report` opens a dashboard with:
- Total cost hero banner
- Summary cards (calls, tokens, cache rate, avg cost, cache savings)
- Top 10 most expensive requests
- Cost breakdown by model
- Cost by day with mini-bar charts
- Recent sessions list

## Requirements

- VS Code ≥ 1.100
- GitHub Copilot extension installed
- Setting enabled: `github.copilot.chat.agentDebugLog.fileLogging.enabled`

## Installation

### Option A — Install the VSIX (no build needed)

```bash
code --install-extension copicost-0.0.1.vsix
```

Or in VS Code: **Cmd+Shift+P** (Mac) / **Ctrl+Shift+P** (Windows/Linux) → `Extensions: Install from VSIX...` → select the file.

Then reload VS Code (**Cmd+Shift+P** / **Ctrl+Shift+P** → `Developer: Reload Window`).

### Option B — Build from source

**Prerequisites:** Node.js ≥ 18, npm

```bash
# 1. Install dependencies
npm install

# 2. Compile TypeScript → out/
npm run compile

# 3a. Run in development (opens a new VS Code window with the extension active)
#     Open the copicost folder in VS Code, then press F5

# 3b. Or package as a .vsix and install it permanently
npx @vscode/vsce package --allow-missing-repository
code --install-extension copicost-0.0.1.vsix
```

> **Tip:** Run `npm run watch` instead of `compile` to recompile automatically on every file save during development.

### Enable the debug logs (required)

CopiCost reads logs that Copilot only writes when this setting is on:

1. Open **Settings** (**Cmd+,** on Mac, **Ctrl+,** on Windows/Linux)
2. Search for `agentDebugLog`
3. Enable **GitHub › Copilot › Chat › Agent Debug Log: File Logging Enabled**

Or add this to your `settings.json`:

```json
"github.copilot.chat.agentDebugLog.fileLogging.enabled": true
```

New sessions will be logged from that point on. Past sessions without logs will not appear.

## Settings

Open **Cmd+,** (Mac) / **Ctrl+,** (Windows/Linux) and search for **`copicost`** to access all options.

| Setting | Type | Default | Description |
|---------|------|---------|-------------|
| `copicost.currency` | `"USD"` \| `"EUR"` | `"USD"` | Currency used to display all costs |
| `copicost.eurUsdRate` | number | `0.92` | EUR/USD rate applied when currency is EUR |
| `copicost.monthlyBudgetUsd` | number | `20` | Monthly budget in USD — used to compute the % in the status bar |

Or add them directly to your `settings.json`:

```json
{
  "copicost.currency": "EUR",
  "copicost.eurUsdRate": 0.92
}
```

> **Note:** All internal calculations are done in USD. The currency setting only affects how amounts are displayed.

## How the Cost is Calculated

### 1. Finding the logs

Copilot writes debug logs under VS Code's workspace storage folder:

```
~/Library/Application Support/Code/User/workspaceStorage/
  <workspace-hash>/
    GitHub.copilot-chat/
      debug-logs/
        <session-id>/
          main.jsonl      ← event stream (one JSON object per line)
          models.json     ← model catalog with pricing
```

CopiCost scans all workspaces automatically — no configuration needed.

### 2. Parsing the event stream

Each line of `main.jsonl` is a JSON span. CopiCost only looks at entries of type `llm_request`, which contain:

```json
{
  "type": "llm_request",
  "attrs": {
    "model": "claude-opus-4.6",
    "inputTokens": 14554,
    "outputTokens": 257,
    "cachedTokens": 9595
  }
}
```

### 3. Pricing source

Prices are read directly from the `models.json` file that Copilot writes alongside each session. They are expressed in **cents per million tokens** and come from the GitHub API at the time of the session — they are always up to date.

Example for Claude Opus 4.6:

| Token type | Price |
|------------|-------|
| Input      | 500¢/M → $5.00/M |
| Output     | 2500¢/M → $25.00/M |
| Cache read | 50¢/M → $0.50/M |

If `models.json` is absent (old sessions), CopiCost falls back to a hardcoded table in `src/config.ts`.

### 4. Formula

For each `llm_request`:

```
fresh_input = inputTokens - cachedTokens

cost = (fresh_input  / 1_000_000) × input_price
     + (cachedTokens / 1_000_000) × cache_price
     + (outputTokens / 1_000_000) × output_price
```

Cached tokens are charged at the (cheaper) cache read rate instead of the full input rate, which is why the cache hit rate directly impacts cost.

## Privacy & Data

**Everything stays on your machine.**

- CopiCost reads only files that VS Code itself already writes locally (`workspaceStorage/`)
- No network requests, no telemetry, no analytics
- No Copilot API calls — pricing comes from the local `models.json`
- Uninstalling the extension leaves no trace; it writes nothing to disk

The logs CopiCost reads are the same ones the built-in **Agent Debug Log panel** displays. They contain prompt content and tool call details — they never leave your machine.
